# Starter React V16 + Redux

+ ```npm install```
+ ```npm start```

Si vous souhaitez générer votre app vous même il vous faudra juste node avec une version >=6 et npm puis :
+ ```npm install -g npx```
+ ```npm install -g react-create-app```
+ ```npx create-react-app votre-nom-d-app```

+ ```cd votre-nom-d-app ```
+ ```npm start```
